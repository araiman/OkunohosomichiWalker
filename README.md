## 概要 
Android向け、歩数計アプリ。2100kmを歩き切ることを目指します。 

**～ウォーキング/毎日の通勤・通学 を少し楽しく～**

## 現在作成した機能 
* 歩数のカウント
* 名前身長の設定 
* 累計の歩数の確認
* 今どこ？
* 全行程確認
